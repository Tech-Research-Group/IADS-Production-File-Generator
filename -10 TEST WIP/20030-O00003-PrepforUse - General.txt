				<title>Introduction</title>
				<para>Assembly and preparation for use of each subsystem in the Expeditionary Platoon Life Support Module (EPLSM) is performed in one or more work packages. This work package serves as a reference guide for setting up the EPLSM. Order listed in this WP maximizes ability of material handling equipment (MHE) to move heavy items, and minimizes MHE interaction with personnel assembling each subsystem. Setup order can be modified based on mission requirement and multiple subsystems may be set up at the same time if enough personnel are available. Refer to <xref figid="O00003-10-5419-218-F001"/> for guidance on laying out site.
					<figure id="O00003-10-5419-218-F001">
						<title>Site Reference Point</title>
						<graphic boardno="G06021801151"/>
					</figure>
				</para>
			</proc>

			<proc>
				<para>Transportation/Storage Subsystem should be set up first in all configurations to allow for setup and support items to be obtained as they are required during assembly of each subsystem. Once storage TRICONs are in position, the priority of each subsystem is left to discretion of unit commander. When modifying setup order, ensure all safety measures are followed.</para>
